07) Descreva o resultado de renderização do HTML abaixo. 

        <html>
        <head>
        <title>Exercício 4 c)</title>
        </head>
        <body>
        <p><a href="index.php">Link 1</a><br>
        <a href="http://www.google.com">Google</a></p>
        <p><a href="http://www.yahoo.com">Yahoo</a> </p>
        </body>
        </html>

R:

        Há três tags <a> no código, que representam um texto clicavel que redirecionará o usuário ao link associado na propriedade href.
        No caso da primeira tag, o link é relativo, ou seja, o arquivo index.php deve estar na mesma pasta que o arquivo que contém o código.
        Já as outras duas tags, possuem links absolutos, ou seja, o link é o endereço completo do site.
        Todas as tags <a> estão dentro de uma tag <p>, que representa um parágrafo.
        A primeira tag <p> possui duas tags <a> dentro, e a segunda possui apenas uma.
        A tag <br> representa uma quebra de linha.
